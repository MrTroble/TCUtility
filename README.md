## How to build this mod

### Setup Eclipse

- ``./gradlew genEclipseRuns eclipse``
- Import project as existing workspace

### Setup IntelliJ IDEA

- ``./gradlew genIntellijRuns``
- Import as gradle project

### Build

- ``./gradlew build``
